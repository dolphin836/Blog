Manjaro Linux 之旅

发表于 2019-01-03 | 更新于 2019-01-04 | 分类于 Linux
前言

接触 Linux 系统始于 2011 年到 2013 年间使用 Fedora 6 开发 QT 应用程序和 C 语言程序，后来写 PHP ，经常捣鼓服务器，又接触了 CentOS。一直想使用 Linux 系统做日常开发工作，先后试过 Fedora、Ubuntu、CentOS、Linux Mint，都不尽满意。直到最近接触 Manjaro，通过近一个月的试用，基本满足了我日常开发的需求。

特色

  • 硬件支持完善
  • 自带内核管理工具
  • 丰富的软件源，包括很多常用的国产软件
  • 丰富的桌面环境，包括 XFCE、KDE、GNOME 等，覆盖需求面广
