在 Github 上搜索编辑器的时候发现了 Quill.js 这个项目，简单的浏览了它的官网，第一感觉就是文档写的非常好。

立马动手按照文档写了一个简单的 Demo，对于一个编辑器我主要关心以下几点：

  • UI 颜值既正义
  • 自定义功能
  • 扩展性

我验证了以下几个常见的需求：

  • 个性化工具栏
  • 图片上传
  • 撤销重做

官网文档都详细介绍了这三个需求的实现方式。

下面是我的 Demo，需要注意的是 撤销/重做 是通过 history 这个 module 实现的。

在 toolbarOptions 中添加了 undo 和 redo 选项，然后通过 Quill 提供的 className 设置它们的样式，我直接给它们设置了一个背景图片。
